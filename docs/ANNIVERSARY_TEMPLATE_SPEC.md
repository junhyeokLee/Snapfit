# ANNIVERSARY Template Spec

- `templateId`: `anniversary_days_v1`
- `templateSlug`: `anniversary_days`
- `category`: `anniversary`
- `theme`: `days`
- `masterRatio`: `portrait`
- `variants`
  - `portrait`: `1080x1440`
  - `square`: `1440x1440`
  - `landscape`: `1440x1080`

## Direction

- `100일`, `200일`, `1주년`처럼 숫자가 중요한 커플 기념일 포토북
- 정보형보다 사진과 짧은 문장이 중심
- `SAVE THE DATE`의 웨딩 카드 톤이 아니라, 더 가볍고 사랑스러운 기념일 앨범 무드
- 큰 숫자 모티프, 스탬프, 리본, 메모 조각을 보조 장식으로 사용

## Visual Language

- 메인 컬러: `cream`, `soft blush`, `paper pink`, `cocoa`
- 포인트 컬러: `cherry rose`, `muted berry`
- 타이포:
  - 제목: `Noto Serif KR`
  - 본문/캡션: `Noto Sans KR`
- 대표 모티프:
  - 큰 숫자 배지
  - 쿠폰/티켓 카드
  - 리본 메모
  - 편지/봉투
  - 로맨틱 문장 카드
- 금지:
  - 웨딩 청첩장처럼 보이는 진한 네이비 카드 반복
  - 제주 템플릿처럼 여행 정보형/지도형 페이지
  - 가족 템플릿처럼 패치형 컬러 블록 위주의 스크랩북 문법
  - 가족 템플릿처럼 메모월, 사이드 밴드, 행잉 포토라인, 네모 그리드 반복
  - 제주 템플릿처럼 세로 밴드, 다크 에디토리얼 프레임, 여행 매거진형 정렬

## Asset Rule

- 무료 이미지 소스 우선순위: `Pexels > Unsplash > Pixabay`
- 예시 이미지는 초기 시안 단계부터 반드시 삽입
- 초기 시안은 `샘플 이미지 1장`을 모든 이미지 슬롯에 반복 적용해도 됨
- 최종 치환 전까지는 이미지 누락 상태로 publish 금지
- preview와 실제 렌더는 같은 JSON 구조를 사용
