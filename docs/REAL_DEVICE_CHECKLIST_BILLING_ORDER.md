# Real Device Checklist (Billing + Order)

## 1) 구독 결제 자동 복귀
1. 앱 실행 -> 마이페이지 -> 구독 및 결제 관리 진입
2. 구독 상품에서 `구독하기` 탭
3. 외부 결제창으로 이동 확인
4. 결제 성공 후 앱으로 복귀되며 `결제 완료가 자동 반영되었습니다.` 스낵바 확인
5. 마이페이지에서 멤버십 상태가 `구독중`으로 변경되는지 확인
6. 저장 용량 카드의 plan/limit이 PRO 기준으로 반영되는지 확인

## 2) 구독 결제 실패/취소
1. 결제창 진입 후 취소 또는 실패 흐름 진행
2. 앱 복귀 시 실패 안내 메시지 노출 확인
3. 구독 상태가 변경되지 않는지 확인

## 3) 제작확정 -> 주문 생성
1. 앨범 리더에서 `...` -> `제작 확정하기`
2. `주문하러가기` 진입
3. 배송지 필수값 입력 후 `주문 생성하기`
4. 주문번호 발급 및 상태 `결제대기` 확인

## 4) 주문 결제 완료 처리
1. 주문 화면에서 `결제 완료 처리` 탭
2. 상태가 `제작중`으로 변경되는지 확인
3. 마이페이지 -> 나의 주문 내역에서 동일 주문 상태 확인

## 5) 주문 상세 UX
1. 주문 카드 탭 -> 상세 바텀시트 노출
2. 주문번호 복사 / 송장 복사 버튼 동작 확인
3. 송장번호가 있으면 `배송 조회` 버튼으로 외부 조회 페이지 열림 확인
4. 상세에서 주소/수령인/타임라인(결제완료, 인쇄접수, 배송시작, 배송완료) 노출 확인

## 6) 알림/상태 동기화
1. 주문 상태 변경 후 주문 내역 새로고침 없이도 복귀 시 반영되는지 확인
2. 마이페이지 주문현황 숫자가 서버 주문상태와 일치하는지 확인

## 7) 회귀 체크
1. 앨범 편집/저장/삭제 동작 영향 없는지
2. 스토어/템플릿 상세/사용하기 동작 영향 없는지
3. 로그인/로그아웃 후 프로필 이미지 유지되는지
