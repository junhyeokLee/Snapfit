# SnapFit 법적/정책 마감 체크리스트

## 1) 앱 노출 문서
- 서비스 이용약관
- 개인정보 처리방침
- 마케팅 수신 동의
- 오픈소스 라이선스

## 2) 운영 문구 확인
- 결제 전: 가격/환불 제한 고지
- 결제 후: 제작 시작 시 취소 제한 고지
- 배송 지연: 알림/고객센터 경로 안내
- 개인정보 문의: 책임자 이메일 노출

## 3) 동의 버전 관리
- 앱 코드:
  - `termsVersion`
  - `privacyVersion`
- 버전 변경 시 로그인 동의 플로우 재확인

## 4) 서버/운영
- 약관/방침 변경일 기록
- CS 대응 템플릿(환불/재제작/배송지연) 준비
- 제3자 위탁 변경 시 문서 즉시 업데이트

## 5) 출시 전 최종 점검
- 최신 문서가 앱의 약관 및 정책 화면에 반영됐는지 확인
- 로그인/재로그인 시 동의 흐름 테스트
- 결제/주문/배송 단계 고지 문구 실제 화면 확인
