# FAMILY_WEEKEND Template Spec

- `templateId`: `family_weekend_v1`
- `templateSlug`: `family_weekend`
- `category`: `family`
- `theme`: `weekend`
- `masterRatio`: `portrait`
- `variants`
  - `portrait`: `1080x1440`
  - `square`: `1440x1440`
  - `landscape`: `1440x1080`

## Direction

- 가족이 함께 보낸 짧은 주말을 담는 사진 중심 앨범
- 정보형 페이지보다 표정, 손짓, 기록물, 붙여 둔 메모가 중심
- `JEJU`와 겹치지 않게 밝은 페이퍼, 패치형 장식, 원형/그리드/메모보드 구성을 사용

## Page Flow

- 최소 `16페이지`

1. cover full bleed
2. family grid
3. note and photo
4. circle story
5. ticket board
6. quote patch
7. four-cut grid
8. wide banner
9. detail mix
10. memory wall
11. ending note
12. big memory
13. polaroid line
14. circle and note
15. banner and detail
16. ending grid

## Asset Rule

- 무료 이미지 소스 우선순위: `Pexels > Unsplash > Pixabay`
- 예시 이미지는 제작 초기에 반드시 삽입
- 미리보기와 실제 렌더는 같은 JSON 구조를 사용
